Exercise: Exploring Different Types of Loops

Objective

Understand how each loop works by writing simple programs that perform the same task using different loops.

Task

Print the numbers from 1 to 5 using each of these loop types.

Part 1: Using a For loop

for i in range(1, 6):
                    print(i)

Part 2: Using a While loop

i = 1
                while i <= 5:
                    print(i)
                    i += 1

Part 3: Using a Do-While loop

Note: Python doesn't have a built-in do-while loop, but in languages like C++, Java, or JavaScript, it looks like this:

int i = 1;
                do {
                    System.out.println(i);
                    i++;
                } while (i <= 5);

In Python (simulating do-while):

i = 1
                while True:
                    print(i)
                    i += 1
                    if i > 5:
                        break

Part 4: Using a For-Each loop

Usually used for iterating over collections like lists:

numbers = [1, 2, 3, 4, 5]
                for num in numbers:
                    print(num)

Discussion Questions

  • What are the differences between these loops?
  • When might you choose one loop over another?
  • Can you modify the examples to count from 5 down to 1?



Exercise: Understanding Loops with a Game

Objective

Learn how loops can be used in a practical scenario by creating a simple game.

Task

Create a number guessing game that uses loops to allow the player multiple attempts to guess the correct number.




Game involving loops

Here's a fun beginner-friendly game in Python that involves loops

It's a simple number guessing game where the player has multiple attempts to guess the secret number. This game will help students understand loops by repeating prompts until the correct guess is made or attempts run out.

Game: Number Guessing Game

import random

                # Generate a random number between 1 and 10
                secret_number = random.randint(1, 10)

                # Set the maximum number of attempts
                max_attempts = 3

                print("Welcome to the Number Guessing Game!")
                print(f"You have {max_attempts} attempts to guess the number between 1 and 10.")

                # Initialize attempt counter
                attempts = 0

                # Loop until attempts are exhausted or the player guesses correctly
                while attempts < max_attempts:
                    guess = int(input("Enter your guess: "))
                    attempts += 1
                    
                    if guess == secret_number:
                        print("Congratulations! You guessed the right number!")
                        break
                    elif guess < secret_number:
                        print("Too low. Try again.")
                    else:
                        print("Too high. Try again.")

                # If the loop ends without a correct guess
                if guess != secret_number:
                    print(f"Sorry, you've used all your attempts. The number was {secret_number}.")

How it involves loops:

  • The while loop allows multiple attempts until the player guesses correctly or runs out of attempts.
  • The loop continues to prompt the user for input, demonstrating repetition.
  • The game can be extended or modified, for example, by increasing attempts, adding hints, or keeping score.

Lesson 2: Applying IT Tools to Solve Problems

Learning Objectives

  1. Apply the most suitable IT tool to complete a given task.
  2. Demonstrate the ability to use digital tools analytically to solve a problem.

Materials

  • Computer lab access
  • Sample task (e.g., "Create a report showing the most efficient way to manage class attendance data")
  • Word processors, spreadsheets, or new web tools

Introduction (15 mins)

Question: Recap of previous lesson — "How do we decide which tool to use?"

Evaluation criteria (use these to compare tools):

  • Speed and performance
  • Features
  • Ease of use / user interface
  • Cost and accessibility
  • File compatibility
  • Collaboration and sharing

Context: Identify a real-world problem (e.g., managing class data or budgeting). Spreadsheets are often suitable because they handle data entry, manipulation, and calculations. Examples include Microsoft Excel, LibreOffice Calc, Airtable, and Google Sheets. Note: Google Sheets excels at real-time collaboration. For demonstration, the teacher will use Excel.

Activity (20 mins)

  • Students form small groups (2–3).
  • Select one IT tool (Excel, Google Sheets, Airtable, or another app).
  • Create a simple solution: attendance tracker, budget calculator, or data summary.
  • Use formulas, charts, or other features to show analytical thinking. Ask the teacher or a peer for help if needed.

Conclusion (5 mins)

Quick reflection: "Why did you choose this tool? What made it effective?"

Assessment

Practical work (10 marks): submit the solution file plus a short written explanation.

Instructions (Practical work)

In groups of two or three, identify a problem and design a solution. Provide:

  1. Problem statement
  2. Solution (file)
  3. Short explanation justifying the choice of software

Sample Problems

  • (a) Students attendance records
  • (b) Budget
  • (c) Simple interest / investment tracker
  • (d) Timetable
